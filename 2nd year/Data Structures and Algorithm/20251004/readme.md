Midterm Exam

Instructions:
<ol>
  <li>Write a function that reverses an array without using built-in reverse functions.</li>
  <li>Use a stack to reverse a string.</li>
  <li>Write a program that checks if a given expression has balanced parentheses using a stack.</li>
  <li>Include all sources used as reference.</li>
</ol>

Example:
<br>
Input: "(a + b) * (c - d)" → Output: Balanced

Input: "(a + b * (c - d)" → Output: Not Balanced
